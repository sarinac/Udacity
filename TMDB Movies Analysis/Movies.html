Introduction¶

I selected the movies dataset available on Kaggle

The dataset consists of ~10,000 movies released on or before July 2017. Data points include cast, crew, plot keywords, budget, revenue, posters, release dates, languages, production companies, countries, TMDB vote counts and vote averages. This dataset also has files containing 26 million ratings from 270,000 users for all 45,000 movies. Ratings are on a scale of 1-5 and have been obtained from the official GroupLens website.

With the movie dataset, I plan on answering the following questions:

  1. Do movies get a higher vote if they are lead by a particular director? What other metrics lead to higher ratings?
  2. Which genres are popular from year to year?
  3. What influences high revenue in movies?

Data Cleaning¶

From observing the data, we need to do the following to clean the data:

  • Drop duplicate values.
  • Fix data points with unrealistic 'runtime', 'revenue', 'budget_adj', and 'revenue_adj'.
  • Break data points with multiple 'genre', 'cast', 'keywords', and 'production_companies' into separate values.
  • Drop columns we don't need for our analysis.

The next step is to break data points with multiple 'genres', 'cast', 'director', 'keywords', and 'production_companies' into separate values. They are separated by a pipe '|'. We will ignore data entries that have null values.

In [37]:
split_columns = ['genres', 'cast', 'keywords','production_companies','director']

for c in split_columns:
    # Get initial set of pipe rows
    pipe_rows = movies[movies[c].str.contains('|',na=False, regex=False)]
    # Drop the original pipe rows
    movies.drop(pipe_rows.index, inplace=True)
    # Append the rows. Rows are no longer separated by pipe
    while pipe_rows.shape[0] > 0:
        first = pipe_rows.copy()
        second = pipe_rows.copy()
        first[c] = first[c].apply(lambda x: x.split("|",1)[0]) # Get value before first pipe
        movies = movies.append(first)
        second[c] = second[c].apply(lambda x: x.split("|",1)[1]) # Get values(s) after first pipe
        second_nopipe = second[second[c].str.contains('|', regex=False) == False]
        movies = movies.append(second_nopipe)
        second_pipe = second[second[c].str.contains('|', regex=False)]
        pipe_rows = second_pipe

Conclusions¶

From this dataset we initially observed that movies from the 1960s to 2010s

In the data wrangling section, we dealt with movies with multiple directors, production companies, and genres. We also learned that the dataset includes not only movies but also shorts and TV Series (indicated by runtime and cross-referencing specific titles on the Internet).

Some limitations we encountered in this dataset include missing or zero values in runtime, budget, and revenue. In order to do our analysis, we made assumptions by replacing these numbers with the median or mean. Fields containing multiple values would only have around 3 or 4 values, which is a subset. For example, Cast would only contain 3 members out of the hundreds of cast in a movie. This dataset also does not contain every single movie since the 1960s, so we have to assume that this data can represent the population of all movies.

In the data exploration section, we learned that movie ratings on a sale of 1 to 10 could vary by director, as there are some directors like Mark Cousins or David Mallet who average above 8. However, ratings aren't strongly influenced by budget or runtime. We also learned that popularity of movies and entertainment have increased over time. They can vary by genre, too. For example, Fantasy have increased significantly in the 2000s, but Horror has not increased as dramatically in popularity. Lastly, we observe that higher revenue is correlated with higher budget and popularity.
